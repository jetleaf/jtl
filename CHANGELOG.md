## [1.0.0]

Initial stable release.

### Added
- JetLeaf Template Language (JTL) core rendering engine for generating HTML, text,
  and dynamic content.
- Template syntax, expression evaluation, and built-in helpers for common
  templating tasks.
- Integration points with JetLeaf runtime and logging packages.

### Notes
- This release marks the first stable version of **JTL**. Any future changes that
  break the public API will include clear migration notes.
