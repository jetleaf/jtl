Replaces the TemplateExpressionEvaluator used by the template engine.

This evaluator handles the execution of expressions within templates, such as conditionals or computed values. Providing a custom evaluator allows support for new operators, complex logic, or integration with external data sources. Changes take effect immediately for all future renders.
