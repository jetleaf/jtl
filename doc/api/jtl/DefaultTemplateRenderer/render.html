Renders the given template using its associated TemplateContext.

  • Parameters

    • template: The Template to render, containing placeholders and structure.
    • context: The TemplateContext providing variable resolution and expression evaluation.
    • built: An already verified or built Asset to skip building later on

  • Returns

    • A SourceCode object containing the fully rendered template, raw content, and optionally parsed structure or metadata.

  • Behavior

    • Performs variable substitution, expression evaluation, and any custom template processing (e.g., filters, includes, loops).
    • Should gracefully handle missing or null values in the context.
    • Implementations may cache intermediate representations or compiled templates to improve performance.

Implementation

@override
SourceCode render(Template template, TemplateContext context, [Asset? built]) {
  final asset = built ?? _assetBuilder.build(template.getLocation());
  final rawContent = asset.getContentAsString();

  // Parse template structure
  final codeStructure = _parseTemplate(rawContent, context);

  // Render template with context
  final renderedContent = _renderTemplate(rawContent, template, context);

  return SourceCodeBuilder()
      .withAsset(asset)
      .withCodeStructure(codeStructure)
      .withRenderedContent(renderedContent)
      .withRawContent(rawContent)
      .build();
}
