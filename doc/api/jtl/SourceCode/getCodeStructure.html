CodeStructure getCodeStructure()

Returns the structured representation of the code.

The CodeStructure captures the hierarchical elements of the code (e.g., HTML tags, JavaScript functions, etc.), enabling introspection, transformations, or analysis before rendering.

Example

final structure = source.getCodeStructure();
for (final element in structure.getElements()) {
  print(element.name);
}
